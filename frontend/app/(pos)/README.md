# POS Terminal Route Group

Full-screen layout for point of sale operations. This route group uses parentheses `(pos)` to enable a dedicated layout without affecting the URL structure.

## Routes

- `/pos` — Main POS terminal interface
- `/pos/history` — Transaction history (future)
- `/pos/reports` — POS reports (future)

## Layout

The POS uses a full-screen layout without the standard ERP sidebar, maximizing screen real estate for retail transactions.
